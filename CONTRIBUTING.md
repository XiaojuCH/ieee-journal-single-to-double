# Contributing

New conversion patterns are welcome, especially small IEEEtran before/after fixes that save authors from common two-column submission traps.

Please keep contributions focused and practical:

1. Fork the repository.
2. Add your pattern to `references/ieee-conversion-patterns.md`.
3. Follow the existing format: a short problem label, a `Before` LaTeX code block, an `After` LaTeX code block, and 1-3 concise rules.
4. Keep examples minimal and self-contained when possible.
5. Open a pull request explaining which IEEEtran conversion issue the pattern fixes.

If your change adds or modifies examples, make sure the LaTeX files compile and the audit script still behaves as expected.
